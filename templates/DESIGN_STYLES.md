# Resume Design Styles - Quick Reference

This document showcases the 5 professional design styles available in the Resume Hack plugin.

## Overview of Styles

| Style | Best For | Color Palette | Key Features |
|-------|----------|---------------|--------------|
| **Modern Tech** | Software engineers, tech roles | Blue accent | Clean, minimal, generous white space |
| **Professional Business** | Finance, consulting, corporate | Navy blue | Traditional, structured, polished |
| **Creative Bold** | Designers, marketers, creative | Purple/vibrant | Bold typography, personality-forward |
| **Minimal Elegant** | Executives, high-level roles | Black/gray | Ultra-minimal, sophisticated |
| **Academic/Research** | Academia, research, scientific | Dark blue | Traditional serif, content-dense |

## Style 1: Modern Tech

### Visual Preview
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
JOHN DOE
Senior Software Engineer
john@email.com | (555) 123-4567 | linkedin.com/in/johndoe
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

EXPERIENCE
▸ Built scalable microservices...
▸ Architected cloud infrastructure...

SKILLS
Languages: Python, JavaScript, Go
Frameworks: React, Django, FastAPI
```

### Characteristics
- **Font**: Inter (sans-serif)
- **Primary Color**: #2563eb (modern blue)
- **Layout**: Left-aligned, single column
- **White Space**: Generous (modern, breathable)
- **Accent**: Blue for headers and bullet points
- **Typography**: Clean hierarchy with bold weights

### Best Use Cases
- Tech startups
- Software engineering roles
- Product management
- Data science
- DevOps/SRE positions

### When to Choose
- Applying to tech companies
- Want modern, professional look
- Need to stand out visually
- Portfolio/GitHub links prominent

## Customization Options

All styles support:

### Color Customization
Change the primary accent color by modifying the CSS variable:
```css
:root {
    --primary: #2563eb; /* Change this */
}
```

### Font Customization
Swap fonts by changing the font-family:
```css
body {
    font-family: 'Your Font', sans-serif;
}
```

### Spacing Adjustments
Tighten or loosen spacing:
```css
body {
    padding: 0.5in; /* Adjust margins */
}

.section {
    margin-bottom: 20px; /* Adjust section spacing */
}
```

## Conversion to PDF

### Recommended Method: Browser Print

1. Open HTML file in Chrome or Firefox
2. Press Ctrl+P (Windows) or Cmd+P (Mac)
3. Select "Save as PDF"
4. Settings:
   - Paper size: Letter or A4
   - Margins: None (built into design) or Default
   - ✅ Background graphics
   - ❌ Headers and footers
5. Save as: `FirstName_LastName_Resume.pdf`

### Quality Check
After conversion:
- ✅ Text is selectable (not image)
- ✅ Colors appear correctly
- ✅ Fits on intended pages (1-2)
- ✅ No content cut off
- ✅ File size <1MB

## ATS Compatibility

All designs maintain ATS compatibility:

✅ **Text-based** (not images)
✅ **Standard HTML** structure
✅ **Selectable text** in PDF
✅ **Logical reading order**
✅ **No complex positioning**

### Testing ATS Compatibility

1. **Copy-paste test**: Can you copy all text?
2. **Text selection**: Is all text selectable in PDF?
3. **Use ATS checker**: Jobscan, Resume Worded
4. **Grayscale test**: Readable without color?

## Design Best Practices

### Do's ✅
- Keep design consistent throughout
- Use adequate white space
- Maintain clear visual hierarchy
- Test print appearance
- Ensure readability at all sizes
- Balance design with content

### Don'ts ❌
- Don't use too many colors (2-3 max)
- Don't make fonts too small (<10pt)
- Don't sacrifice readability for design
- Don't use images or photos (US resumes)
- Don't over-design for conservative industries
- Don't ignore ATS compatibility

## Quick Selection Quiz

**Answer these questions to find your style:**

1. **What industry?**
   - Tech → Modern Tech
   - Finance → Professional Business
   - Creative → Creative Bold
   - Executive → Minimal Elegant
   - Academic → Academic/Research

2. **Company culture?**
   - Startup → Modern Tech
   - Corporate → Professional Business
   - Agency → Creative Bold
   - Conservative → Professional Business

3. **Your personality preference?**
   - Modern, clean → Modern Tech
   - Traditional, polished → Professional Business
   - Bold, creative → Creative Bold
   - Sophisticated, minimal → Minimal Elegant
   - Academic, detailed → Academic/Research

4. **Career level?**
   - Early career → Modern Tech or Professional Business
   - Mid-career → Any style that fits industry
   - Senior → Professional Business or Minimal Elegant
   - Executive → Minimal Elegant
